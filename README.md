# 📊 Sistema de Préstamos MVC

Sistema web desarrollado en ASP.NET Core MVC para la gestión de préstamos a empleados, incluyendo control de pagos, intereses, autorizaciones y historial de movimientos.

---

## 🧱 Tecnologías utilizadas

- ASP.NET Core MVC
- C# (.NET)
- ADO.NET
- SQL Server
- HTML5 / CSS3
- Bootstrap 5
- JavaScript
- SweetAlert2

---

## 🎯 Funcionalidades principales

### 📌 Módulo de Préstamos
- Alta de préstamos a empleados
- Cálculo de montos e intereses
- Control de saldo pendiente
- Fechas de pago personalizadas

### 👥 Gestión de Empleados
- Selección de empleados para préstamo
- Empleados autorizadores

### 💳 Tipos de pago
- Semanal
- Catorcenal
- Quincenal
- Mensual
- Al final

### 🧾 Historial de cambios
- Registro automático de modificaciones
- Auditoría de préstamos (quién, cuándo y qué cambió)

### 📊 Dashboard
- Total de préstamos
- Total prestado
- Total por cobrar
- Saldo global

## 🗄️ Base de datos

1. Abrir SQL Server Management Studio
2. Ejecutar el script ubicado en: carpeta de Database  BdPrestamosWeb.sql
   
3. Verificar creación de tablas:
- Prestamos
- Empleados
- TiposPagos
- PrestamosHistorico
